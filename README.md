# Serverless FrontEnd

Installed npm as

```bash
curl -sL https://deb.nodesource.com/setup_16.x | sudo -E bash -

sudo apt install nodejs
```

```bash
sudo npm i -g yarn


yarn create next-app --experimental-app
```

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
